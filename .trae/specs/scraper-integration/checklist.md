# Checklist

## 代码实现验证

- [ ] 抓取模块能正确访问 GitHub 并解析 HTML
- [ ] 重试处理器在失败时正确等待 5 分钟后重试
- [ ] 重试处理器在达到最大次数后停止重试
- [ ] 定时任务在指定时间触发（日报 8:00、周报 8:30、月报 9:00）
- [ ] 抓取成功后自动保存到 data/briefs/{type}/ 目录
- [ ] 抓取成功后**立即**自动触发 AI 分析
- [ ] AI 分析完成后**立即**自动生成 HTML 报告
- [ ] HTML 报告生成后**立即**自动更新首页
- [ ] HTML 报告生成完成后**立即**发送成功推送通知
- [ ] 连续失败 12 次后发送失败通知
- [ ] 完整流程一体化执行（抓取 → AI → HTML → 推送）

🔥 热度指数：4/5
多个项目围绕 Claude Code 生态（如 gsd-build/get-shit-done、shareAI-lab/learn-claude-code）形成技术集群，但物理引擎与地图生成等细分领域仍保持理性发展

  • gsd-build/get-shit-done：36k stars 的元提示工程系统，重塑 Claude Code 开发范式

推荐建议

  • 集成 opendataloader-project/opendataloader-pdf 到 RAG 系统，提升非结构化文档处理能力
  • 采用 gsd-build/get-shit-done 的规范驱动开发模式优化 Claude Code 工作流

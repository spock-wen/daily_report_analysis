# Agent

## 域描述
智能体（Agent）是能够自主执行任务、做出决策并与环境交互的AI系统。本域包括各种智能体框架、多智能体系统和智能体应用。

## 相关子领域
- multi-agent: 多智能体系统
- orchestration: 智能体编排
- self-improvement: 自我改进智能体
- memory: 智能体记忆管理
- chatbot: 聊天机器人
- trading: 交易智能体
- scientific: 科学研究智能体
- security: 安全智能体

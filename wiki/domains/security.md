# Security

## 域描述
安全领域涉及网络安全、渗透测试、逆向工程等安全相关技术。本域包括安全工具、安全分析和安全测试系统。

## 相关子领域
- cybersecurity: 网络安全
- penetration-testing: 渗透测试
- reverse-engineering: 逆向工程
- api-extraction: API提取
- osint: 开源情报
- reconnaissance: 侦察
- information gathering: 信息收集

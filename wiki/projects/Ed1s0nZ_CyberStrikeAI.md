## 核心功能
- AI 原生渗透测试平台，用 Go 语言实现
- 集成 100+ 安全工具，覆盖完整攻击链
- 智能编排引擎，支持多步骤任务规划和执行
- 基于角色的测试，预定义安全角色
- 技能系统，具备专门的测试技能
- 全面的生命周期管理能力
- 原生 MCP 协议，支持 HTTP/stdio 以及外部 MCP 接入
- 兼容 OpenAI/DeepSeek/Claude 等模型的智能决策引擎
- Web 控制台，支持可视化攻击链、风险打分与步骤回放
- Web 登录保护、审计日志、SQLite 持久化
- 大结果分页、压缩与全文检索
- 支持工具编排与扩展，通过 YAML 定义命令、参数、提示词与元数据
- 网络扫描、Web 应用扫描、漏洞扫描、子域名枚举、API 安全、容器安全、云安全、二进制分析、漏洞利用、密码破解、取证分析、后渗透等多种功能

## 核心功能
- 内置自进化学习循环，从经验中创建技能并持续改进
- 分层持久记忆系统，跨会话保持记忆和用户画像
- 支持200+主流AI模型，无供应商锁定
- 15+消息平台全接入，包括Telegram、Discord、Slack、飞书、钉钉等
- 企业级安全体系，包括指令审批、Docker容器沙箱隔离等
- 本地Web仪表盘，用于管理Hermes Agent
- Fast Mode，优先处理OpenAI和Anthropic模型
- 子代理并行处理复杂工作流
- 上下文富集功能，包括搜索预览、多语言文件提示、压缩检查点
- 支持Termux/Android原生运行
- 背景进程监控，实时通知匹配模式
- 完整的备份和恢复功能

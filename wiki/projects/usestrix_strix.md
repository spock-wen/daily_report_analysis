## 核心功能
- 开源 AI 黑客工具，自动发现并修复应用漏洞
- 自主 AI 代理，像真正的黑客一样运行代码、发现漏洞并通过实际概念验证进行验证
- 完整的黑客工具包，包括 HTTP 代理、浏览器自动化、终端环境、Python 运行时、侦察和代码分析
- 全面的漏洞检测，包括访问控制、注入攻击、服务器端漏洞、客户端漏洞、业务逻辑漏洞、认证问题和基础设施配置错误
- 多代理编排，实现分布式工作流、可扩展测试和动态协调
- 与 GitHub Actions 和 CI/CD 管道无缝集成
- 自动修复和报告功能，加速修复过程
- 支持扫描本地代码库、GitHub 仓库和黑盒 web 应用评估
- 提供 Strix 全栈安全平台，支持连接代码库和域名，分钟内启动渗透测试
